Beginning Groovy and Grails Chapter 1 Code

Chapter 1 compares and contracts Groovy and Java so the
code shows a simple example implemented in both Groovy and Java.

To run the Java example type the following:

>javac com\apress\bgg\Todo.java
>java com.apress.bgg.Todo

To run the Groovy example type the following:

>groovy com\apress\bgg\Todos.groovy
